DIVCNT2 题解

题目大意

求∑i=1nσ(i2)\sum_{i=1}^n\sigma(i^2)∑i=1n​σ(i2)，此处σ(i)\sigma(i)σ(i)表示iii的约数个数。

n≤1012n\leq 10^{12}n≤1012

解题思路

先考虑化简 σ(x2)\sigma(x^2)σ(x2)：

设 x=p1c1⋅p2c2⋯pkckx = p_1^{c_1}\cdot p_2^{c_2}\cdots p_k^{c_k}x=p1c1​​⋅p2c2​​⋯pkck​​，则 x2=p12c1⋅p22c2⋯pk2ckx^2 = p_1^{2c_1}\cdot p_2^{2c_2}\cdots p_k^{2c_k}x2=p12c1​​⋅p22c2​​⋯pk2ck​​。

σ(x2)=(2c1+1)⋅(2c2+1)⋯(2ck+1)\sigma(x^2) = (2c_1+1)\cdot (2c_2+1)\cdots (2c_k+1)σ(x2)=(2c1​+1)⋅(2c2​+1)⋯(2ck​+1)

好像化不下去了。（不过题解第一篇好像就是这样化的，有时间研究一下）

换一种化法：

∑i=1n∑j∣i∑k∣i[gcd⁡(j,k)=1]\sum_{i=1}^n\sum_{j|i}\sum_{k|i}[\gcd(j,k) = 1]∑i=1n​∑j∣i​∑k∣i​[gcd(j,k)=1]

提出 jjj、kkk：

∑j=1n∑k=1n[gcd⁡(j,k)=1]∑j∣i,k∣i1\sum_{j=1}^n\sum_{k=1}^n[\gcd(j,k) = 1]\sum_{j|i,k|i}1∑j=1n​∑k=1n​[gcd(j,k)=1]∑j∣i,k∣i​1

由于只有 gcd⁡(j,k)=1\gcd(j,k) = 1gcd(j,k)=1 时才有贡献，所以：

∑j=1n∑k=1n[gcd⁡(j,k)=1]∑jk∣i1\sum_{j=1}^n\sum_{k=1}^n[\gcd(j,k) = 1]\sum_{jk|i}1∑j=1n​∑k=1n​[gcd(j,k)=1]∑jk∣i​1

后面的东西可以化简：

∑j=1n∑k=1n[gcd⁡(j,k)=1]njk\sum_{j=1}^n\sum_{k=1}^n[\gcd(j,k) = 1]\frac{n}{jk}∑j=1n​∑k=1n​[gcd(j,k)=1]jkn​

之后，令 T=dkT = dkT=dk：

∑T=1nnT∑j∣T[gcd⁡(j,Tj)=1]\sum_{T=1}^n\frac{n}{T}\sum_{j|T}[\gcd(j,\frac{T}{j}) = 1]∑T=1n​Tn​∑j∣T​[gcd(j,jT​)=1]

反演：

∑T=1nnT∑j∣T∑d∣gcd⁡(j,Tj)μ(d)\sum_{T=1}^n\frac{n}{T}\sum_{j|T}\sum_{d|\gcd(j,\frac{T}{j})}\mu(d)∑T=1n​Tn​∑j∣T​∑d∣gcd(j,jT​)​μ(d)

枚举 ddd：

∑d=1nμ(d)∑T=1nnT∑j∣T,d∣Tj1\sum_{d=1}^n\mu(d)\sum_{T=1}^{n}\frac{n}{T}\sum_{j|T,d|\frac{T}{j}}1∑d=1n​μ(d)∑T=1n​Tn​∑j∣T,d∣jT​​1

发现 jjj、Tj\frac{T}{j}jT​ 都要满足是 ddd 的倍数，就先给 TTT 两个 ddd ，之后就只用计算 TTT 的约数个数就行了（此处的 σ(T)\sigma(T)σ(T) 和题目中意思相同）：

∑d=1nμ(d)∑T=1nd2nTd2σ(T)\sum_{d=1}^{n}\mu(d)\sum_{T=1}^{\frac{n}{d^2}}\frac{n}{Td^2}\sigma(T)∑d=1n​μ(d)∑T=1d2n​​Td2n​σ(T)

整除分块后，前面的 μ\muμ 最大不超过 10610^6106，故可以直接预处理出来，只用考虑后面的东西怎么求就行了：

∑i=1nniσ(i)\sum_{i=1}^n\frac{n}{i}\sigma(i)∑i=1n​in​σ(i)

对前面的系数整除分块，后面的 σ\sigmaσ 很大的时候可以使用整除分块求解，加一个记忆化就是杜教筛的复杂度了（我不会分析，但是看上去长得就差不多嘛）。

由于这个 nd2\frac{n}{d^2}d2n​ 非常小，所以复杂度不会太离谱。

总时间复杂度为 O(n23)O(n^{\frac{2}{3}})O(n32​)，有亿点卡常。

总结

莫比乌斯反演中考的无非就是约数啊，gcd⁡\gcdgcd 啊，约数个数啊，这要求你对它们的性质了如指掌。

比如这道题中，第一步的变形就是约数个数和中的常见套路，之后的不断更换求和指标也是常用技巧。

当出现 i⋅ji\cdot ji⋅j 时，要用 TTT 把它代替，这样才能有更多限制。

不要遗漏式子中任何重要的信息，即使是小小的 gcd⁡(i,j)=1\gcd(i,j) = 1gcd(i,j)=1 也包含 iii、jjj 互质这一重要条件。

推式子时要想清楚式子的意义，在推不通时想更多的思路，更换更好的求和指标，最后才能到达成功的彼岸。

式子即哲学，结果即真理。
