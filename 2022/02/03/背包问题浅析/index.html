背包问题浅析

发表于 2022-02-03 更新于 2022-03-15 Valine：

最基础的背包——01背包

问题：

给定一个容量为www的背包，有nnn件物品，每件物品都有一个价值vvv和一个体积ccc，问背包能装下的物品的最大价值和

思路：

每件物品都有选或不选(0或1)两种状态，考虑用此作为dpdpdp的一个维度，对于特定容量的价值，是从之前的容量的价值递推出来的，故可以用此作为dpdpdp的第二个维度，于是就有了dpdpdp数组f[i][j]f[i][j]f[i][j]表示考虑了前iii个物品，容量为jjj时的最大价值

如何转移呢？

对fff数组的修改必然涉及到一个变量：价值vvv，所以需要vvv参与dpdpdp的转移方程，稍加考虑即可推出一个简明的方程：

f[i][j]=max(f[i−1][j],f[i−1][j−c[i]]+v[i])f[i][j] = max(f[i-1][j],f[i-1][j-c[i]]+v[i])f[i][j]=max(f[i−1][j],f[i−1][j−c[i]]+v[i])

maxmaxmax的第一项表示不选择这个物品，直接使用i−1i-1i−1个物品的最大价值；第二项表示选择这个物品，所获得的价值是不放这个物品的背包的最大价值加上这个物品的价值

可以发现，这个方程只和第iii项和第i−1i-1i−1项有关，所以可以使用滚动数组做空间优化，即f[j]=max(f[j],f[j−c[i]]+v[i])f[j] = max(f[j],f[j-c[i]]+v[i])f[j]=max(f[j],f[j−c[i]]+v[i])

但是，需要注意此处可能对f[j]f[j]f[j]进行修改，但是我们需要用的是修改之前的值，所以需要倒序修改f[j]f[j]f[j]，防止f[j−c[i]]f[j-c[i]]f[j−c[i]]变成了现在的价值，而不是在放入i−1i-1i−1个物品的价值

时间复杂度：O(nw)O(nw)O(nw)

题目：P1048 采药

01背包的变形——完全背包

问题：

给定一个容量为www的背包，有nnn种物品，每种物品都有一个价值vvv和一个体积ccc，且数量无限，问背包能装下的物品的最大价值和

思路：

还是考虑原来的dpdpdp方程：

f[j]=max(f[j],f[j−c[i]]+v[i])f[j] = max(f[j],f[j-c[i]]+v[i])f[j]=max(f[j],f[j−c[i]]+v[i])

把修改顺序变为正序修改，思考一下f[j]f[j]f[j]的意义

当j′=j+c[i]j' = j+c[i]j′=j+c[i]时，f[j′]=f[j]+v[i]f[j'] = f[j]+v[i]f[j′]=f[j]+v[i]，此时的f[j]f[j]f[j]表示的是考虑前iii个物品的最大值，显然当前物品放入的情况也被考虑进去了，所以此时正序递推正好满足题目要求

时间复杂度：O(nw)O(nw)O(nw)

优化后的01背包——多重背包

问题：

给定一个容量为www的背包，有nnn种物品，每种物品都有一个价值vvv和一个体积ccc，且数量为mmm个，问背包能装下的物品的最大价值和

因为每种物品数量有限，所以无法用完全背包求解（可能会使用多于mmm个物品），考虑将其转化为01背包

最朴素的想法是把每种物品拆成mmm个单独的物品，把所有的物品都单独拿出来，这样就变成了01背包问题

但是，这样做的复杂度是O(nmw)O(nmw)O(nmw)的，在n≥1000n \geq 1000n≥1000时就跑不过了，所以要考虑优化

由于二进制的很多特殊性质，以及对logloglog的谜之追求，出现了一种优化方法：二进制拆分

众所周知，任何一个数都可以被表示为二进制形式

例如：$ 42 = (101010)_2$

所以，我们把这mmm个物品拆成一堆二进制，例如上面的424242可以拆成(1)2=1(1)_2=1(1)2​=1,(10)2=2(10)_2=2(10)2​=2,(100)2=4(100)_2=4(100)2​=4,(1000)2=8(1000)_2=8(1000)2​=8,(10000)2=16(10000)_2=16(10000)2​=16,最后剩下来一个111111。

不难看出，前面555个二进制可以组成0∼310 \sim 310∼31中的任何一个数，而其中的32∼4232 \sim 4232∼42中的所有数都可以由21∼3121 \sim 3121∼31加上111111得到，所以0∼420 \sim 420∼42中的所有数都可以由这些拆出来的数组合得到

这样，我们可以把每个背包拆成这些物品，之后当作01背包求解，于是复杂度就降低到了O(nlog⁡mw)O(n \log mw)O(nlogmw)

题目：P6567 买表

融合的01背包——分组背包

问题：

给你nnn个物品，每个物品属于一个组，同一组中只能选择一个物品，问容量为mmm的背包的最大价值

思路：

同一组中的所有物品都只能选一次，我们通过01背包来限制选择的次数

同一组中的所有物品只能选一个，我们通过同一价值来避免一个组中选择多个物品

时间复杂度：O(nm)O(nm)O(nm)
